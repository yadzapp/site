# YADZ Site
Website for the app YADZ, Yet Another DayZ Launcher. Simple app for launching DayZ. 

[yadz.app](https://yadz.app/)

## Setup
```bash
# Install dependecies
yarn install

## Start development server on `localhost:3000`
yarn dev

## Build the application for production
yarn build
```
